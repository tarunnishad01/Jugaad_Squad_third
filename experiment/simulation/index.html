Welcome to the interactive Simulink experiment on testing linearity and non-linearity of systems. In this lab, you will use Simulink blocks to simulate different systems and determine whether they are linear or non-linear. First, choose a system block. You can select a Gain or Integrator block for a linear system, or try using a Saturation or Relay block to simulate non-linear behavior. Next, choose your input signal. You may use a Step input, Sine wave, or Ramp signal. Now, run the simulation and observe the output waveform using the Scope block. Perform the additivity test by applying two input signals and check if the output equals the sum of the outputs of individual signals. Then, perform the homogeneity test by scaling the input and observing if the output is proportionally scaled. If both tests are satisfied, the system is linear. If not, the system is non-linear. This experiment is fundamental for understanding system behavior in control and signal processing.
